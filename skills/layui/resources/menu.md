# 基础菜单

## 作用

静态菜单结构，基于 `dropdown` 组件驱动，适合应用菜单、上下文菜单和分组动作菜单。

## 结构要点

- 容器：`<ul class="layui-menu"></ul>`
- 菜单组、父级菜单、分隔线
- 支持 `lay-options`

## 菜单结构

| 节点 | 说明 |
| --- | --- |
| 菜单项 | 普通点击项 |
| 分隔线 | 视觉分组 |
| 父级项 | 展开子菜单 |
| 组标题 | 分组说明 |

## 事件

- `dropdown.on('click(filter)', callback)`

## AI 使用规则

- 菜单适合静态结构或少量动态项，不适合复杂表单交互。
- 如果需要多级展开，应说明父子关系和点击行为。
