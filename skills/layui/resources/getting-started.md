# 开始使用

## 作用

这是 Layui 2 的入口页，用于说明如何下载、引入和快速运行第一个示例。

## 引入

标准资源结构：

- `layui/css/layui.css`
- `layui/layui.js`

也可使用 npm、GitHub Releases 或 CDN。

## 快速上手

最小示例通常包含：

1. `<!DOCTYPE html>`
2. 引入 `layui.css`
3. 引入 `layui.js`
4. 使用 `layui.use(function(){ ... })`
5. 访问模块，例如 `layui.layer`
6. 调用 API，例如 `layer.msg('Hello World')`
