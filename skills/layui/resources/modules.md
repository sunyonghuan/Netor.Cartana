# 模块系统

## 作用

Layui 的模块系统用于定义、使用、扩展和组织组件能力，是页面脚本组织的核心机制。

## 核心概念

- 模块挂载在 `layui` 对象下
- `layui.define()` 用于定义模块
- `layui.use()` 用于加载模块
- `layui.extend()` 用于扩展外部模块

## 用法要点

- 可直接使用内置模块，如 `layui.layer`、`layui.form`
- 支持定义业务入口模块
- 支持按页面加载不同业务模块

## 常见结构

| 项目 | 说明 |
| --- | --- |
| 定义模块 | `layui.define` |
| 引用模块 | `layui.use` |
| 扩展模块 | `layui.extend` |
| 导出能力 | `exports` |

## AI 使用规则

- 页面脚本应优先用模块化方式组织，不要在全局散写大量逻辑。
- 自定义模块名称应与业务域对应，避免和官方模块冲突。
