# 评分组件

## 作用

用于评分、等级、星级展示，适合商品评价、内容打分和满意度反馈。

## API

- `rate.render(options)`

## 常用属性

| 属性 | 说明 |
| --- | --- |
| `elem` | 容器 |
| `length` | 星级总数 |
| `value` | 默认值 |
| `half` | 是否允许半星 |
| `theme` | 主题色 |
| `text` | 说明文案 |
| `readonly` | 只读 |
| `setText` | 文案回调 |
| `choose` | 选中回调 |

## 常见场景

- 商品评分
- 满意度调查
- 用户等级展示

## AI 使用规则

- 评分组件应明确是否可编辑还是仅展示。
- 文案应与分值联动，避免只展示星星没有说明。
