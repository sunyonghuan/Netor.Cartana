# 颜色

## 作用

提供 Layui 的基础色板和主题色使用说明，用于页面文本、背景、边框、提示和状态表达。

## 色系

- 基色调：主色与核心品牌色
- 辅色调：danger / warning / success / info
- 中色调：灰阶与中性色
- 七色调：常用页面配色基底

## 用途

- 文本
- 背景
- 边框
- 状态提示

## 建议用法

| 类型 | 场景 |
| --- | --- |
| 主色 | 主按钮、关键强调 |
| 成功色 | 成功状态、通过提示 |
| 警告色 | 风险提示、注意事项 |
| 危险色 | 删除、错误、异常 |
| 信息色 | 中性提示、说明 |

## AI 使用规则

- 颜色应服务于信息层级，不要仅凭个人审美随意堆色。
- 状态色要保持统一，不同页面保持同一语义映射。
