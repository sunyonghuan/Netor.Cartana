# 面板 / Card / Collapse

## 作用

用于承载页面内容的容器组件，覆盖普通面板、卡片、折叠面板、手风琴等常见信息组织方式。

## 内容

- 常规面板
- 卡片面板
- 折叠面板
- 手风琴
- 嵌套折叠
- 渲染与事件

## 常见类

| 类名 | 说明 |
| --- | --- |
| `layui-panel` | 基础面板 |
| `layui-card` | 卡片容器 |
| `layui-collapse` | 折叠容器 |
| `layui-colla-item` | 折叠项 |

## 使用建议

- 卡片适合展示单块信息。
- 折叠面板适合节省空间的分组内容。
- 嵌套折叠适合分层配置，但不要过深。

## AI 使用规则

- 面板类组件主要用于结构，不要把交互逻辑塞进容器本身。
- 折叠内容里如果嵌入表单，渲染后需重新初始化子组件。
