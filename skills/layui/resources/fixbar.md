# 固定条

## 作用

页面滚动时固定在一侧的工具条，适合返回顶部、反馈、客服和快捷操作。

## API

- `util.fixbar(options)`

## 常用属性

| 属性 | 说明 |
| --- | --- |
| `bars` | 图标列表 |
| `default` | 默认行为 |
| `bgcolor` | 背景色 |
| `css` | 自定义样式 |
| `target` | 触发目标 |
| `scroll` | 滚动阈值 |
| `margin` | 边距 |
| `duration` | 动画时长 |
| `on` | 点击回调 |

## 常见场景

- 返回顶部
- 反馈入口
- 联系客服
- 页面快捷按钮

## AI 使用规则

- 固定条只适合轻量操作，不要塞复杂表单。
- 若点击后要执行业务动作，应明确每个图标的含义与行为。
