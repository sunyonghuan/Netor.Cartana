# 服务管理子技能

## 功能描述
管理服务器上的系统服务，包括启动、停止、重启、状态查看。

## 使用场景
- 服务故障恢复
- 服务配置更新后重启
- 定期检查服务状态
- 批量服务管理

## 支持操作
- 启动服务
- 停止服务
- 重启服务
- 查看服务状态
- 设置开机自启
- 查看服务日志

## 支持的服务管理工具
- systemd (CentOS 7+/Ubuntu 16+)
- init.d (旧版本系统)
- supervisor
- Docker 容器

## 安全注意
- 关键服务操作需二次确认
- 操作前记录当前状态
- 提供回滚方案
- 记录操作日志
