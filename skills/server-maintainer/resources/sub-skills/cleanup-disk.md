# 清理磁盘子技能

## 功能描述
清理服务器磁盘空间，包括临时文件、缓存文件、旧日志等。

## 使用场景
- 磁盘空间不足预警
- 定期维护清理
- 应用部署前清理

## 操作步骤
1. 连接目标服务器
2. 分析磁盘使用情况
3. 识别可清理文件
4. 执行清理操作
5. 验证清理结果

## 清理目标
- /tmp 目录
- /var/tmp 目录
- /var/log 旧日志
- 包管理器缓存
- 应用缓存

## 安全注意
- 不删除 7 天内的日志
- 不删除系统关键文件
- 清理前创建快照（如支持）
- 记录清理日志
