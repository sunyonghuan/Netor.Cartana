# 安全扫描子技能

## 功能描述
扫描服务器安全漏洞和配置问题，提供修复建议。

## 使用场景
- 定期安全检查
- 新服务器上线前检查
- 安全事件后排查
- 合规性检查

## 扫描内容
- 系统漏洞
- 弱密码检测
- 开放端口检查
- 防火墙配置
- 文件权限检查
- 日志安全分析
- 恶意软件扫描

## 扫描工具
- Lynis（系统安全审计）
- Chkrootkit（Rootkit 检测）
- Fail2ban 状态检查
- SSH 配置检查

## 输出报告
- 风险等级分类
- 详细问题描述
- 修复建议
- 优先级排序

## 安全注意
- 扫描不影响业务运行
- 敏感信息脱敏
- 报告加密存储
- 及时修复高危问题
