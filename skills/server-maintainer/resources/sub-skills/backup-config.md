# 配置文件备份子技能

## 功能描述
备份服务器重要配置文件，支持定时备份和恢复。

## 使用场景
- 系统变更前备份
- 定期配置备份
- 灾难恢复准备
- 配置版本管理

## 备份目标
- Nginx/Apache 配置
- 数据库配置
- 应用配置文件
- 系统关键配置
- 防火墙规则

## 备份策略
- 全量备份（每周）
- 增量备份（每日）
- 保留策略（30 天）
- 异地备份（可选）

## 恢复操作
- 列出可用备份
- 选择备份版本
- 执行恢复
- 验证恢复结果

## 安全注意
- 备份文件加密
- 敏感信息脱敏
- 备份权限控制
- 定期测试恢复
