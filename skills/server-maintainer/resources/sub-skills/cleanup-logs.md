# 清理日志子技能

## 功能描述
清理服务器各类日志文件，释放磁盘空间。

## 使用场景
- 日志文件过大
- 定期日志轮转
- 系统维护

## 日志类型
- 系统日志 (/var/log/syslog, /var/log/messages)
- 应用日志
- Web 服务器日志 (Nginx/Apache)
- 数据库日志
- 安全日志

## 清理策略
- 保留最近 7 天日志
- 压缩 7-30 天日志
- 删除 30 天前日志
- 日志轮转配置

## 安全注意
- 清理前备份重要日志
- 不删除当前正在写入的日志
- 记录清理操作审计日志
