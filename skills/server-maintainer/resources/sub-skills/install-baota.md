# 安装宝塔面板子技能

## 功能描述
在 Linux 服务器上安装宝塔面板，提供可视化服务器管理界面。

## 使用场景
- 新服务器初始化
- 需要可视化面板管理
- 快速部署 Web 环境

## 安装版本
- 最新版（默认）
- 指定版本（用户指定）

## 安装步骤
1. 检查系统环境（CentOS/Ubuntu/Debian）
2. 下载宝塔安装脚本
3. 执行安装脚本
4. 获取默认账号密码
5. 配置安全组/防火墙

## 安装后配置
- 修改默认端口
- 创建管理员账号
- 配置 SSL 证书
- 安装常用插件

## 安全注意
- 修改默认登录地址
- 设置强密码
- 配置防火墙规则
- 定期更新面板
